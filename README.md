<h1>Student Management System With DataBase</h1>
<p>StudentManagement is an console app that help you out creating a student database that is out-saved in a JSON file</p>
<br>
<p> Lets say that you have a school or you are the person incharged of managing a student data in a school. StudentManagement allow you: </p>

- <p>Creating a student database with, id, name, age, program coursed, status (active/inactive)</p>
- <p>Store the information in Json file that allow you keep using the created data even after stop running the app.</p>
- <p> Allows to detele and updating every single student information.</p>

<h2>How it works?</h2>
<p>StudentManagement is splited in four python files every one of them with an important function in the console application: </p>


<h2>Files Function</h2>

- <h3>messages.py</h3> <p>This is the simples one, it take care of the styled message printed across the app, like the welcome message, the error messages and so on!<p>

- <h3>validation.py</h3> <p>This file takes care of the validation error and imports the sytled error messages from messages.py to look comfortable to the user</p>

- <h3>student_management.py</h3> <p>This is the principal file in the app because it has the class that make the application works, basically the CRUD system</p>

- <h3>main.py</h3> <p>The main.py file is the place in which we put all the functions together in a loop and so the system runs until the user decide to stop it</p>


<h2>Language</h2>

<p>The languange used is Python splited into the four files that interact by themselve across the application</p>
